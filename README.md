# Godot Minigame 4

## Devlog

For one, I kinda forgot that NPCs don't actually pick up balls, since they only spawn on the player side. Mine was pretty close to the example, I just combined some of the actions onto one arrow. I didn't think about the things like throw power, jump power, throw frequency. I learned that I could use an is on ground check to change the color of the ball and possibly to make it safe.
